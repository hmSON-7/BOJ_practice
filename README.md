# BOJ_practice

백준 온라인 저지( BOJ )에서 제공되는 문제들을 풀어 모아두는 리포지토리입니다.  
모든 디렉토리는 **삼성 SW 역량 테스트 대비 기출**, **Solved.ac 클래스별**, **Non‑Class**로 분류되며,  
각 분류 안에서 Solved.ac 난이도(랭크)별로 세부 정리되어 있습니다.  
*※ 실버 5 이상 문제만 게시합니다.*
